## Running
- Modify Makefile.
  - CADICAL_INC should find src of CaDiCaL, and CADICAL_LIB_DIR its build.
- To measure resistance:
  - make full
  - ./full <input> [options]
- To convert to DNF:
  - make to_dnf
  - ./to_dnf <CNF>
